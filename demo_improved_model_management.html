🎯 使用指南

基础配置步骤

  1. 打开系统设置页面
  2. 点击"模型配置"选项卡
  3. 配置模型平台信息
  4. 设置嵌入模型参数
  5. 配置重排序模型
  6. 保存配置

高级功能使用

  1. 查看模型状态和指标
  2. 执行健康检查
  3. 动态切换活跃模型
  4. 监控性能数据
  5. 处理异常情况
  6. 优化配置参数

🔧 技术架构

前端架构
Settings Page → ModelManager → API Client → UI Components
后端架构
Config API Router → Model Management → Configuration Storage
数据流
用户操作 → 前端处理 → API调用 → 后端处理 → 配置更新 → 状态同步
